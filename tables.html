HTTP状态码对照表

服务器响应状态码及其含义说明

1xx 信息性状态码 - 请求已被服务器接收，继续处理

100
Continue

服务器收到请求的初始部分，客户端应继续发送请求

101
Switching Protocols

服务器同意切换协议（如HTTP切换到WebSocket）

102
Processing

服务器正在处理请求，但尚未给出响应

2xx 成功状态码 - 请求已成功被服务器接收、理解、并接受

200
OK

请求成功，常用于GET和POST请求

201
Created

请求已成功，并创建了新的资源

202
Accepted

请求已接收，但尚未处理完成

204
No Content

请求成功，但返回的响应没有内容

206
Partial Content

服务器成功返回部分内容（用于断点续传）

3xx 重定向状态码 - 需要进一步操作以完成请求

301
Moved Permanently

永久重定向，请求的资源已永久移动到新位置

302
Found

临时重定向，资源暂时移动到新位置

303
See Other

重定向到其他资源（使用GET请求）

304
Not Modified

资源未修改，可以使用缓存

307
Temporary Redirect

临时重定向，POST方法不会改变

308
Permanent Redirect

永久重定向，POST方法不会改变

4xx 客户端错误状态码 - 请求包含语法错误或无法完成请求

400
Bad Request

服务器无法理解请求的语法

401
Unauthorized

请求需要用户认证（未登录）

403
Forbidden

服务器拒绝访问（无权限）

404
Not Found

服务器找不到请求的资源

405
Method Not Allowed

请求方法不被允许

408
Request Timeout

请求超时

409
Conflict

请求与服务器状态冲突

410
Gone

资源已被永久删除

413
Payload Too Large

请求体过大

414
URI Too Long

请求的URI过长

415
Unsupported Media Type

不支持的媒体类型

429
Too Many Requests

请求过于频繁，触发限流

5xx 服务器错误状态码 - 服务器在处理请求时发生错误

500
Internal Server Error

服务器内部错误

501
Not Implemented

服务器不支持请求的功能

502
Bad Gateway

网关或代理服务器收到无效响应

503
Service Unavailable

服务器暂时不可用（维护或过载）

504
Gateway Timeout

网关或代理服务器超时

505
HTTP Version Not Supported

HTTP协议版本不支持
