📖 使用说明

输出格式选择：

  • 字典 (Dict) - 生成Python字典字面量，适合直接使用
  • 类 (Class) - 生成普通Python类，支持__init__方法
  • Pydantic模型 - 生成Pydantic BaseModel，支持数据验证
  • Dataclass - 生成@dataclass装饰器类

Python类型映射规则：

  • str - 字符串类型
  • int - 整数类型
  • float - 浮点数类型
  • bool - 布尔类型
  • list[T] - 列表类型
  • dict[str, T] - 字典类型
  • Optional[T] - 可空类型
  • datetime - ISO日期格式

命名转换示例：

  • userName → user_name (snake_case)
  • user_name → userName (camelCase)
  • createdAt → created_at (snake_case)
