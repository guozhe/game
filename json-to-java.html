📖 使用说明

功能特点：

  • 支持嵌套对象自动生成内部类
  • 支持数组和列表类型自动识别
  • 支持Lombok注解 (@Data, @NoArgsConstructor, @AllArgsConstructor)
  • 支持serialVersionUID自动生成
  • 支持多种字段命名风格转换

Java类型映射规则：

  • String - 字符串类型
  • Integer / Long - 整数类型 (根据数值大小自动选择)
  • Double / BigDecimal - 小数类型
  • Boolean - 布尔类型
  • List<T> - 数组类型
  • Object - 复杂嵌套对象

命名转换示例：

  • user_name → userName (驼峰)
  • user_name → UserName (帕斯卡)
  • userName → user_name (下划线)
